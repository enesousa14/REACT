# 💼 Portfólio React

Este é meu portfólio profissional desenvolvido com React, com o objetivo de apresentar meus projetos e habilidades como desenvolvedora.

---

## 🚀 Tecnologias utilizadas

- React
- JavaScript
- HTML5
- CSS3

---

## 📂 Estrutura do projeto

src/
 ├── components/
 │    ├── Header.jsx
 │    ├── Sobre.jsx
 │    └── Projetos.jsx
 ├── data/
 │    └── projetos.js
 ├── App.js
 └── App.css

---

## 🧩 Funcionalidades

- Estrutura baseada em componentes (React)
- Seção "Sobre mim"
- Seção "Projetos"
- Renderização dinâmica de projetos utilizando array de objetos
- Layout simples e responsivo
